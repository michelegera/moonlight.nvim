- Initial release of moonlight.nvim
- Authentic Moonlight VSCode color palette
- Full TreeSitter support (legacy and modern @ syntax)
- LSP diagnostics and references highlighting
- Git signs integration
- Extensive plugin support:

  - WhichKey keybinding hints
  - BufferLine buffer/tab line
  - Dashboard and Alpha start screens
  - Lualine statusline theme
  - Neo-tree and nvim-tree file explorers
  - nvim-cmp completion menu
  - nvim-notify notifications
  - Trouble diagnostics list
  - Aerial code outline
  - Hop and Leap motion plugins
  - Rainbow delimiters
  - IndentBlankline guides
